What can you try in this playground?

  • Toolbar configuration — toggle individual buttons or swap to a custom toolbar layout
  • Themes — switch between light, dark, and auto (system preference) themes
  • Markdown shortcuts — type **bold**, # Heading, - list and watch them transform instantly
  • @mention — type @ to trigger the mention autocomplete
  • Export — download the editor content as HTML or Markdown
  • Bubble toolbar — select any text to see the floating context toolbar

AutumnNote is a zero-dependency WYSIWYG rich text editor built with vanilla JavaScript. Read the full API docs →
